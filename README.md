# webpack 简单配置
vue 的 单页面 的webpack 配置

# 为什么
1. vue-cli 3.0 隐藏了配置文件
2. 用来学习 webpack的配置
3. 满足公司项目需要
# 支持目标
- [] vue 单文件组件
- [] 最新依赖，包括vue、webpack等
- [] 开发模式：热更新
- [] css scoped 生产样式 抽出 
- [] font png file-loaders
- [] svg 支持symbol使用icon
- [] 生产模式优化：公共chunk等
- [] 构建速度优化：dll
- [] 完全使用typescript
- [] 单元测试：mocha + chai
- [] e2e测试：cypress
- [] 编辑器配置：vscode配置、 prettier、editor
